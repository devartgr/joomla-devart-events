Professional event management solution for Joomla 6, designed for municipalities, organizations, associations, conferences, cultural institutions, educational organizations, publishers, business websites, and high-performance event portals.

## Overview

DevArt Events is a modern Joomla 6 event management solution built for creating, managing and publishing professional event listings with support for multiple event dates, venues, calendars, maps, structured data, and SEO.

The extension is designed for production environments and high-traffic Joomla websites while maintaining a lightweight frontend and modern Joomla architecture.

The package includes a Joomla component and frontend module.

---

## Features

### Event Management

Create and manage professional events with:

- Multiple event dates
- Single-day events
- Multi-day events
- Recurring date support
- Featured events
- Event images
- Event descriptions
- Contact information
- Website links
- Social links

---

### Event Views

Built-in event views:

- All Events
- Today Events
- Upcoming Events
- Past Events
- Featured Events

Frontend ordering is based entirely on event dates rather than article creation or publish dates.

---

### Categories & Tags

Organize events using:

- Categories
- Tags
- Featured events
- Joomla ACL
- Joomla publishing workflow

Suitable for very large event collections.

---

### Venue Management

Professional venue management including:

- Saved venues
- Manual venues
- Address management
- City
- State
- Country
- Postal code
- Latitude & Longitude
- Venue logo
- Map picker

Events can use either saved venues or manually entered locations.

### Maps

### Menu Items

Includes Joomla menu items for:

- Event List

### Frontend Module

Display event lists anywhere in Joomla.

Supported display modes:

- Today
- Upcoming
- Past
- Featured
- All Events

Uses the same event-date engine as the component for consistent results.

---

### Event Date Engine

DevArt Events uses an event-based date engine.

Frontend filtering and ordering are based on actual event dates rather than article creation or publishing dates.

Examples:

- Today → Today's event schedule
- Upcoming → Next available event date
- Past → Latest completed events
- All Events → Latest event dates first
